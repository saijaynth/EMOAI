# Emo AI UI Flow

## Core User Journey
1. Landing screen with value proposition
2. Language selection (required)
3. Mood input method picker
4. Input collection screen
5. Analysis loading screen with animation
6. Results screen with song cards
7. Feedback actions and playlist save

## Screen Notes
### Landing
- Bold headline, gradient background, animated sound-shapes.
- CTA: Start Mood Session

### Language Select
- Prominent list/grid of languages.
- Remember last used language.

### Mood Method Hub
- Cards for Text, Voice, Face, Quiz.
- Show reliability hint per method.

### Results
- Mood chip with confidence.
- Song cards: title, artist, language, mood tags.
- Refinement controls: energy slider, context chips.

## Accessibility and UX
- Clear focus states
- Keyboard support for all interactions
- Reduced-motion mode fallback
- High-contrast text over dynamic backgrounds
